-----------------------------------------
Resources for Estonian Constraint Grammar (CG) based Syntactic Analyzer 
-----------------------------------------

 This folder contains rule and lexicon files for Estonian Constraint Grammar based Syntactic Analyzer.
 Files from different sources are combined:
 
 1) Lexicon files from  https://github.com/EstSyntax/EstCG/tree/467f0746ae870169776b0ed7aef8825730fea671 :
    abileksikon06utf.lx
    tmorftrtabel.txt
 
 2) Rule files from  http://math.ut.ee/~tiinapl/CGParser.tar.gz
    clo_ub.rle
    morfyhe_ub.rle
    PhVerbs_ub.rle
    pindsyn_ub.rle
    strukt_ub.rle
    Rules are expected to be compatible with VISL CG3 version 0.9.9.10993.
